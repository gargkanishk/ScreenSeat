My Movie Manager

My Movie Manager is a movie venue and show booking application built with Flask.

______________________________________

Requirements
To run this application, you will need:

Python 3.x
Flask
Pandas
SQLAlchemy
Matplotlib
Installation
Clone or download the project repository.

___________________________________________

Install the required Python packages using pip:

pip install -r requirements.txt

____________________________________________

Running the Application

Make sure that all files (including the database file "database.db") are in the same folder.

Open a terminal or command prompt window, and navigate to the folder containing the project files("code" folder)

(all flask code is in "app.py")


Run the Flask application:

export FLASK_APP=app.py
flask run

Open a web browser and navigate to http://localhost:5000 to view the application.

___________________________________________

Usage

Once you have the application running, you can use it to book venues and shows. The specific functionality of the application will depend on your implementation.
